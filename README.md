# Nooly 🧶

Nooly é uma plataforma web que conecta pessoas que produzem **artesanato** (crochê, tricô, punch needle, bordado, etc.) a **clientes interessados em peças feitas à mão**, de forma personalizada e próxima.

A ideia é valorizar o trabalho artesanal, facilitar a comunicação entre clientes e artesãs, e criar uma **comunidade de artesãs** com perfis tipo Instagram, onde elas podem postar seus trabalhos e receber pedidos.

---

## 🎯 Objetivo do Projeto

- Dar visibilidade e oportunidades de renda para artesãs.
- Permitir que clientes solicitem peças personalizadas e escolham profissionais por localização ou estilo.
- Criar uma comunidade de artesãs com portfólio próprio.
- Oferecer uma experiência segura, simples e agradável para ambos os perfis de usuários.

---

## 📦 Funcionalidades do MVP

### Para o Cliente:
- Cadastro e login.
- Visualização de perfis de artesãs.
- Envio de mensagens diretas ou propostas.

### Para a Artesã (Profissional):
- Cadastro com foto, bio e localização.
- Criação e edição de portfólio (upload de fotos das peças).
- Sistema de mensagens para negociação.

### Comum:
- Feed de trabalhos das artesãs (estilo Instagram).
- Sistema de mensagens simples em tempo real.
- Configuração de limite de postagens/propostas para planos gratuitos/premium.

---

## 🛠 Tecnologias

- **Frontend:** Next.js  
- **Backend / Autenticação / Banco:** Supabase (PostgreSQL + Auth + Storage)  
- **Hospedagem:** Vercel  
- **Armazenamento de imagens:** Supabase Storage  
- **Estilo:** Tailwind CSS
